# ElvUI_SwingBar - Wrath of the Lich King (3.3.5a)	



ElvUI_SwingBar is a plugin for [ElvUI](https://github.com/ElvUI-WotLK/ElvUI) which adds a bar which allows for easy tracking of melee swings
